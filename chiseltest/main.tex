\section{Pympress 简介}

\subsection{Pympress 是什么}

\begin{frame}
  \frametitle{Pympress 是什么}

  Pympress 是一款 PDF 演示工具，专为双屏设置（例如演示和公开演讲）而设计。高度可配置、功能齐全且便携。

  \begin{itemize}
    \item 支持嵌入式 gif（开箱即用）、视频和音频（通过 VLC 或 Gstreamer 集成）
    \item 演示者窗口中显示的文本注释。
    \item 原生支持第二屏幕上的 Beamer 笔记以及 Libreoffice 笔记页面！
  \end{itemize}
\end{frame}

\subsection{Pympress 安装}

\begin{frame}
  \frametitle{Pympress 安装}
  支持 Ubuntu、Debian、Fedora、CentOS 、Mageia 、OpenSuse、RHEL、Arch Linux、macOS、Windows，参见 \href{https://github.com/Cimbali/pympress?tab=readme-ov-file\#installing-}{Installing}
\end{frame}

\subsection{Latex 指令}

\begin{frame}[fragile]
  \frametitle{适用于 Beamer 的 \LaTeX 指令}
  \begin{codeblock}{\LaTeX notes 代码}
\setbeameroption{show notes on second screen}

  \end{codeblock}
\end{frame}

\subsection{运行 Pympress}

\begin{frame}
  \frametitle{Pympress 功能}

  \begin{itemize}
    \item 双屏显示
    \item 媒体支持
    \item 高亮显示模式
    \item 转到幻灯片
    \item 幻灯片概览
    \item 软件指针
    \item 演讲时间细分
    \item 自动文件重新加载
    \item 自动全屏
  \end{itemize}
   \note[item]{双屏显示：在笔记本电脑或平板电脑上查看当前幻灯片、下一张幻灯片、通话时间和挂钟时间以及注释（PDF 注释、第二张幻灯片上的投影仪注释或 Libreoffice 注释页面）。 投影仪或 Libreoffice 注释在幻灯片中的位置会自动检测，并且可以通过菜单选项覆盖。}
   \note[item]{媒体支持：支持播放嵌入（或链接自）PDF 文件的视频、音频和 gif 文件，具有可选的开始/结束时间和循环播放。}
   \note[item]{高亮显示模式：允许用户在当前屏幕上的幻灯片上徒手绘制。按住 ctrl 的同时单击幻灯片（在任一窗口中）将在幻灯片上显示软件激光指针。或者按 L 永久打开激光笔。}
\end{frame}
