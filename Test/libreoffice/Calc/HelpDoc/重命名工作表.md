1. 双击表格名（sheet tab），选择重命名
2. 输入新名称后点击确定
3. 也可以按住Alt后点击表名直接修改
工作表名称可以包含几乎任何字符。有一些命名限制，下述字符不允许出现在工作表名称中:

    colon :

    back slash \

    forward slash /

    question mark ?

    asterisk *

    left square bracket [

    right square bracket ]

    single quote ' as the first or last character of the name
