# 重命名工作表  
使用帮助; https://help.libreoffice.org/7.3/zh-CN/text/scalc/guide/rename_table.html?&DbPAR=CALC&System=UNIX  
说明： 此帮助文档部分为英文，实际上操作步骤为中文，不影响阅读，英文部分讲的是命名限制的字符  
部分翻译：  
1. 双击表格名（sheet tab），选择重命名
2. 输入新名称后点击确定
3. 也可以按住Alt后点击表名直接修改
